Topics and terminology

After much consideration, a few terms were selected for the launch of this site. While there are only a few from the beginning, the ideas are all important factors when making design choices. We hope to add more with time!

Each page is designed with the content in mind, and features examples and some interactive elements to demonstrate the concepts.

Accessibility

Accessibility is a principle which relates to the usability of a system for any and all users. Here, we discuss the topics of physical and visual impairments and their relationship with designing systems.

Consistency

Consistency within a system refers to aesthetic, functional, internal, and external similarities between things like elements and objects. We learn a bit about some considerations for aesthetic consistencies in design and functional consistencies in systems.
